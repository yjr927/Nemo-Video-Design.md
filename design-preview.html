Do's and Don'ts

08

Do

  • Use ship/code tokens as the production source of truth.
  • Keep brand blue scarce and meaningful.
  • Support light and dark mode through semantic tokens.
  • Use Figma values for state intent and anatomy.

Don't

  • Replace #3369FF with Figma #165DFF without migration.
  • Use decorative gradients as UI component colors.
  • Hardcode hex values when tokens exist.
  • Invent component variants like primaryBlue.
